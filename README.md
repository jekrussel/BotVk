# Vk Bot for .NET

# План развития

- [ ] Авторизацию бота.
- [ ] Оповещение о начале работы бота.
- [ ] Отправка личных сообщений.
- [ ] Механизм ответов на комментарии.
- [ ] Интеграция с базой ответов бота.
- [ ] Анализ комментариев/сообщений и выбор наиболее подходящего ответа.
